# 🚕 Taxi Dispatch System

Real-time Taxi Dispatch & Fleet Management System built using Python, Streamlit, Plotly, and Folium. The project simulates intelligent cab allocation with smart dispatching algorithms, demand heatmaps, live analytics, ETA prediction, and dynamic fare calculation.

---

## ✨ Features

- 🚖 Smart cab dispatching using Quick Sort
- 🔍 Linear Search for available taxis
- 🗺️ Interactive heatmaps with Folium
- 📊 Real-time analytics dashboard
- 💰 Dynamic fare calculation with surge pricing
- ⏱️ ETA prediction with traffic simulation
- ⭐ Driver ratings and performance tracking
- 🚕 Fleet management console
- 📈 Ride history and demand analysis

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Folium
- Streamlit-Folium

---

## 🧠 Algorithms Used

- Quick Sort
- Linear Search
- Haversine Distance Formula
